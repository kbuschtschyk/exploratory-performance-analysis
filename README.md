# exploratory-performance-analysis
This project focused on collecting, structuring, and analyzing multi-file movie datasets sourced from Wikipedia JSON files.  The objective was to consolidate fragmented raw data into a unified analytical dataset and perform exploratory data analysis (EDA) to identify trends, distributions, and structural relationships within the film industry data.  
*My Tasks*:  
Retrieved and combined multiple JSON files into a unified structured dataset.  
Performed data cleaning, normalization, and feature preparation using Pandas.  
Conducted exploratory data analysis (EDA) to examine distributions and trends.  
Built visualizations to evaluate production patterns and feature relationships.  
*Result*:  
Successfully transformed fragmented raw JSON files into a consolidated analytical dataset.  
Identified key distribution patterns and structural characteristics in the dataset.  
Strengthened capability in data wrangling, transformation, and visualization workflows.
